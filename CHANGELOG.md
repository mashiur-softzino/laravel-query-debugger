# Changelog

All notable changes to `laravel-query-debugger` will be documented in this file.

## 1.0.0 - 2024-XX-XX

- Initial release
- Auto-detect N+1 query problems
- Identify slow queries with configurable thresholds
- Detect duplicate queries
- Generate actionable fix suggestions with code examples
- Export reports to JSON and HTML formats
- Context tracking (file and line number)
- Beautiful console output with Laravel components
- Performance metrics and statistics
- Automatic query logging via middleware
- Facade for programmatic access
- Production safety features
- Comprehensive test suite with Pest
